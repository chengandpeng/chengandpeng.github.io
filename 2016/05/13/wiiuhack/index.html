WiiU 5.5/5.5.1 破解

发表于 2016-05-13   |  

题外话

自从node升了v6, hexo就沦陷了，到现在还没能支持。实在等不了了，只能node降级了。
当然也可以用nvm安装多个node版本，不过因为以前用RVM做ruby版本控制有点阴影，所以就罢了，反正一个版本也够用了。

WiiU破解

本破解教程参考自FlimFlam69’s 5.5.1 Kernel Exploit Guide (And other various things)

确认WiiU版本

之前可以破解的最新版本是5.3.2， 现在可以将WiiU先升级到5.5.1版本。
如果看文章时最新版本已经超过5.5.1，请先确认是否支持破解再升级。

屏蔽自动升级

为了防止任天堂升级内核补漏洞，需要将WiiU的自动升级屏蔽，有以下两个方式

1.修改DNS

在WiiU的Internet设置中修改DNS相关设置

  1. Turn Auto-Obtain DNS OFF
  2. Set Primary DNS to 107.211.140.065
  3. Set Secondary DNS to 107.211.140.065

修改完DNS以后，WiiU无法解析服务器地址，就无法自动升级了，但是也登陆不了eShop了，原贴中有相关解决方案。同时也看不了Youtube，这对墙内的同学来说也没什么区别。

2.屏蔽URL

类似修改hosts文件，在路由器中将如下地址给屏蔽掉

  • nus.cdn.c.shop.nintendowifi.net
  • nus.cdn.shop.wii.com
  • nus.cdn.wup.shop.nintendo.net
  • nus.wup.shop.nintendo.net
  • nus.c.shop.nintendowifi.net
  • c.shop.nintendowifi.net
  • cbvc.cdn.nintendo.net
  • cbvc.nintendo.net

因为是在路由器中屏蔽的，如果同时有3DS或者其他任天堂机器连在同一个wifi中，也一样登陆不了相关商店。

关闭自动升级

注意在View Mac Address下面还有一个Automatic Software Downloads的选项注意设置为Disable。

配置SD卡

现在破解只支持SD卡，还不支持外置硬盘，所以需要购置一块SD卡。
据说支持128G SDXC，我觉得没必要，用的是64G SDXC TF卡（插卡套），确认可支持。

  1. 格式化SD卡为FAT32。（Mac下做格式化会有问题，我是通过vmware虚拟windows做的）
  2. 下载最新 Homebrew Launcher
  3. 下载最新 Loadiine GX2
  4. 解压上面下载的2个文件，将文件对应以下路径拷贝到SD卡中
  • SD:/wiiu/apps/homebrew_launcher/homebrew_launcher.elf
  • SD:/wiiu/apps/homebrew_launcher/icon.png
  • SD:/wiiu/apps/homebrew_launcher/meta.xml
  • SD:/wiiu/apps/loadiine_gx2/loadiine_gx2.elf

下载的游戏放在如下路径
SD:/wiiu/games/Example Game [GAMEID]
如果下载的游戏没有GAMEID，可以去GameTDB查询。
游戏资源自己网上搜吧。

启动Kernel Exploit

  1. 将SD卡插入WiiU卡槽
  2. 打开WiiU,打开内置浏览器
  3. 删除Cookies(Delete Cookies)
  4. 打开地址http://loadiine.ovh/ （建议加入收藏夹）
  5. 国内打开可能有点慢，页面加载完毕后点击下拉菜单右边的绿色按钮。
  6. 如果之前的配置都没有问题，等待片刻（也可能比较久），就会进入Homebrew.(如果白屏不动了请重启WiiU再次操作)

配置启动服务器

以后每次我们想玩的时候，必须打开http://loadiine.ovh/，但是因为网络条件，可能耗费的时间比较多，这时候我们可以自己搭设服务器来启动Homebrew.

网上有用XAMPP在Windows下架服务器的教程，有兴趣的可以去尝试。

Mac的话也差不多，有空研究一下更新文章。

最后

5.5/5.5.1版本的破解其实去年底就有消息了，拖到3月份发布者消息后也就不了了之了。
之后上个月任天堂发布NX的发售日程，WiiU的停产消息。
现在WiiU失守，另一(yin)方(mou)面(lun)也可以看做帮了任天堂，将WiiU变废为宝。相信接下来一段时间WiiU的销售会有一阵小高潮。
不过Loadiine目前还不支持online play，所以也无法体验到好多以在线为主的游戏。
最后还是希望大家支持正版，哈哈。
