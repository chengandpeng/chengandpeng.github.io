少一点套路，多一点真诚

发表于 2016-06-02   |  

最近霉运当头，记录一下权当释放情绪了…

诚恒

我的中学母校：江苏省南通中学，有个校训叫“诚恒”—-诚于做人，恒于学问。
在中国残酷的应试教育体系，特别是竞争激烈的江苏省境内，我不记得除了做考卷，学校在素质教育上有投入什么心血。
同时对我们那个年纪的人来说，传统的说教式素质教育往往会带来反效果，越是满口仁义道德，越可能引发他人杀人放火。这跟当时社会总体认识，价值观冲突有比较大关系，深层次我也不懂，就不说了。
总之这“诚恒”的校训也就靠建了棟名为“诚恒楼”的教学楼，让大家铭记于心，至少我是这样。
–“你在哪栋楼上课啊？” –“诚恒楼。”
–“小兄弟，诚恒楼在啊？” –“前面左拐右手第一栋就是。”

什么是诚于做人？没有老师做过过多解释。也可能在全校大会上校长念过一段冗长的文章来诠释过我们的校训，
可因为知道这些东西考试也不加分，所以对我们来说都是些过耳云烟。

而当时我们的做人原则就是丛林法则，谁牛逼谁就是正义，吃亏了别抱怨，等你强大了你也能这样对别人。

新一代视频播放器：MPV

发表于 2016-03-08   |  

为什么要用MPV

视频播放器一向是各个平台上比较重要的一个工具软件。特别对于中国，在这个下载比较自由的国家，我们对于播放器的要求一向比较高，既要支持古老的RMVB格式，又要解的了H.265，还要能友好处理UTF-8，GBK，GB2312编码的字幕，着实比较麻烦。

Windows

从KMPlayer，到PotPlayer。分离渲染自带，也方便添加修改，算是Windows下比较好的解决方式，现在发展到什么样子我也不太清楚了。

Mac

Mac下的播放器选择不多但也不乏精品，可多少有些缺陷。

  • 早期安装Perian解码包让QuickTime支持更多格式，但是字幕支持太差。
  • VLC算是比较全面的一个了，解码快，支持几乎大部分编码和格式，但是对ASS的中文字幕支持太差，各种方格子简直就是让你边看电影边玩猜猜看游戏。。srt的字幕倒是没问题，但有时候就偏偏只有ASS,或者就想用ASS的时候，就没法用了。而且软件也是越来越臃肿，打开文件的速度也是越来越慢。
  • MplayerX是一款国人出品，以Mplayer为基础开发的一款轻量级播放器，因为长久不更新，渲染器老旧，播放265的时候，各种雪花甚是崩溃，但是必须说的是，它对中文字幕的支持，以及CPU占用，文件打开速度之快，还是很占优势的。是我之前使用频率最高的播放器。

虽说各有缺点，但是VLC和MplayerX也能覆盖各种场景了。然而某天，这两个都搞不定了。
一个声音为6/7 channel的DTS，封装为MKV的文件，用HDMI输出到电视，MplayerX无法将声音从HDMI输出到电视上，搜索了半天，貌似MplayerX在处理5.1(7.1)声道时，无法正确解析并通过HDMI传送。
VLC是没问题，但是有ASS的方框问题啊。
废话不多说，还是用MPV吧。

发表于 2016-02-22   |   分类于 生活   |  

Hope it’s the last first

已经记不清这是我写的第几篇“first blog”了。
从学会上网开始，就免不了想在网上留下一些自己的印记。从最初的QQ空间开始，胡乱的记录一些中二气息爆棚的生活感想。
摆脱臃肿的Qzone（中二），用上了当时彰显“高端”的MSN的个人空间（后来改名为Live Spaces），简约不失大气的风格让我坚持又叨叨了几年的生活琐事，直到10年，微软宣布停止Live Spaces服务，迁移到WordPress网站。等文章迁移完毕，图片丢的丢，排版乱的乱，坚持则写了一段时间，想想还是自己搞个服务器弄博客吧。

那个时候还没什么VPS，在Godaddy上买了个域名，顺道租了个服务器。一键搭建WordPress，至此开始新的篇章，好景不长，繁重的学业加上懒惰的个性，等到要续费的时候借着省钱的借口就停下了博客的脚步（正好当时兴起Twitter，Goolge Wave<–夭折的货），走上了微博客的道路。
